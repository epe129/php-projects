# PHP Projects Workspace

This workspace contains several small PHP projects and demo applications.

## Projects

- `blog-platform`
  - A blog platform demo with a Python Flask backend and a PHP frontend.
  - Includes backend scripts for database creation, admin management, and a simple web interface.

- `login-system`
  - A login and registration system built with PHP.
  - Includes frontend pages for login and main dashboard, and backend handlers for authentication and registration.

- `quote`
  - A small PHP project for displaying quotes.
  - Simple demonstration app with a single `index.php` entry point.

- `vibe-code`
  - A task and schedule management demo using PHP.
  - Contains pages for login, registration, dashboard, adding tasks, and generating schedules.

## Notes

- Each project is self-contained in its own folder.
- Run each application from its project directory if you need to test or develop locally.
